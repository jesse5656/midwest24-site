# Midwest24 Core Branding Standards

## Overview

This directory contains all branding assets for Midwest24 Core.

Midwest24 Core is a sovereign contractor modernization platform focused on:

- Security
- Data Ownership
- Workforce Development
- Business Continuity
- Contractor Modernization

Brand assets are organized to separate editable source files from production-ready website assets.

---

# Directory Structure

```
branding/
├── source/
├── icons/
├── midwest24-logo.png
├── midwest24-logo-light.png
├── midwest24-logo-full.png
├── midwest24-favicon.png
└── README.md
```

---

# Source Files

Editable source files are stored in:

```
branding/source/
```

Examples:

```
midwest24-logo-master.xcf
midwest24-logo-light.xcf
midwest24-favicon.xcf
```

These files should be retained for future modifications.

Source files are not required for website deployment.

# Logo Rules

## Allowed

✓ Blue shield

✓ White wordmark on dark backgrounds

✓ Transparent backgrounds

✓ Soft white glow for dark hero sections

✓ Consistent aspect ratios

---

## Not Allowed

✗ Stretching logos

✗ Changing shield proportions

✗ Recoloring shield outside approved palette

✗ Heavy shadows

✗ Busy backgrounds behind logo

✗ Adding effects not included in approved assets

# Website Optimization Standards

PNG exports:

- Compression: 9
- Automatic Pixel Format
- Save Color Profile
- Transparent Background

Target file size:

```
100 KB - 300 KB
```

```
500 KB
```
